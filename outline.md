# Raychel智能闹钟广告网站项目大纲

## 项目概述
创建一个以HER电影为灵感的智能闹钟广告网站，通过两个核心场景展示产品的情感价值和技术特性。

## 网站结构

### 1. 首页 (index.html)
- **Hero区域**：电影级视觉效果的沉浸式体验
- **场景选择器**："入睡场景"和"唤醒场景"的切换体验
- **产品预览**：智能闹钟的核心功能展示
- **情感连接**：通过视觉和音频建立用户情感共鸣

### 2. 剧本页面 (script.html)
- **完整剧本展示**：两个场景的详细对话和动作描述
- **分镜展示**：视觉化的场景分解
- **角色介绍**：Raychel和Theodore的人物设定
- **创作背景**：广告创意的来源和理念

### 3. 产品功能页面 (features.html)
- **技术规格**：详细的产品参数和功能
- **交互体验**：可操作的模拟器展示
- **个性化设置**：用户可体验的定制功能
- **购买信息**：产品获取方式

## 核心功能模块

### 视觉效果系统
- 动态粒子效果（萤火虫、流星）
- 光影投射模拟
- 音频可视化
- 平滑的场景过渡

### 交互体验系统
- 场景切换器
- 语音播放控制
- 产品3D预览
- 个性化设置体验

### 内容管理系统
- 剧本文本管理
- 音频文件管理
- 图像资源管理
- 动画效果管理

## 技术实现

### 前端技术栈
- HTML5 + CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- Anime.js (动画效果)
- Pixi.js (粒子系统)
- Shader-park (高级视觉效果)

### 媒体资源
- 生成的hero图像
- 产品展示图像
- 投影效果图像
- 音频文件（语音样本）

### 响应式设计
- 移动端优化
- 平板设备适配
- 桌面端完整体验

## 用户体验流程

1. **首页沉浸** → 用户被电影级视觉效果吸引
2. **场景探索** → 通过切换器体验两个核心场景
3. **功能了解** → 深入了解产品技术特性
4. **情感连接** → 通过个性化体验建立情感纽带
5. **购买决策** → 通过综合体验促进购买转化
