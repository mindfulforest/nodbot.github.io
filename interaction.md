# 智能闹钟Raychel广告网站交互设计

## 核心交互体验

### 1. 场景选择器
- 用户可以在"入睡场景"和"唤醒场景"之间切换
- 每个场景都有对应的视觉效果和音频体验
- 切换时播放平滑的过渡动画

### 2. Raychel语音体验
- 用户可以点击播放Raychel的语音样本
- 展示不同的语调：温柔陪伴、轻柔唤醒、日常对话
- 语音播放时显示音频频谱可视化效果

### 3. 产品功能探索
- 交互式产品3D视图，用户可以旋转查看智能闹钟
- 点击不同部件显示详细功能说明
- 灯光效果模拟器：用户可以调节色温和亮度
- 投影效果预览：森林剪影和流星动画

### 4. 个性化设置体验
- 用户可以输入自己的名字，体验个性化的唤醒对话
- 选择不同的唤醒音乐和灯光组合
- 预览不同的睡眠场景设置

## 用户体验流程

1. **首页沉浸体验** - 用户进入网站即看到电影级的hero场景
2. **场景探索** - 通过场景选择器体验两个核心场景
3. **功能了解** - 通过交互式产品展示深入了解功能
4. **个性化体验** - 尝试个性化设置，建立情感连接
5. **购买决策** - 通过情感体验和功能了解促进购买

## 交互特色

- 所有交互都有平滑的动画过渡
- 使用电影级的视觉效果营造氛围
- 音频体验与视觉效果完美结合
- 响应式设计确保所有设备上的优质体验
